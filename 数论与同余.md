# 数论与同余

## 概述
数论是研究整数性质的数学分支，同余理论是数论中的重要工具，在密码学、计算机科学和编码理论中有广泛应用。

## 整数的表示

### 基b展开
设b>1是整数。正整数n可以唯一地表示为：
n = aₖbᵏ + aₖ₋₁bᵏ⁻¹ + ··· + a₁b + a₀

其中k是非负整数，aᵢ是小于b的非负整数。n的表示称为n的基b展开，记作(aₖaₖ₋₁...a₁a₀)₆。

### 常见进制
- **二进制**: (101011111)₂ = 2⁸ + 2⁶ + 2⁴ + 2³ + 2² + 2¹ + 2⁰ = 351
- **八进制**: (7016)₈ = 7·8³ + 1·8 + 6 = 3598
- **十六进制**: 二进制、八进制、十六进制之间的转换很容易

### 基b展开算法
要构造整数n的基b展开：
1. 将n除以b得到n = bq₀ + a₀，其中0 ≤ a₀ < b
2. 余数a₀是n的基b展开中最右边的数字
3. 将q₀除以b得到q₀ = bq₁ + a₁，其中0 ≤ a₁ < b
4. a₁是从右边数的第二个数字
5. 继续依次将商除以b，直到商为0

## 整数运算的复杂度

### 二进制加法
a = (aₙ₋₁aₙ₋₂...a₁a₀)₂, b = (bₙ₋₁bₙ₋₂...b₁b₀)₂
- 时间复杂度: O(n)位加法

### 二进制乘法
a = (aₙ₋₁aₙ₋₂...a₁a₀)₂, b = (bₙ₋₁bₙ₋₂...b₁b₀)₂
ab = a(b₀2⁰ + b₁2¹ + ··· + bₙ₋₁2ⁿ⁻¹)
- 时间复杂度: O(n²)移位和O(n²)位加法

### 二进制模幂运算
依次找到b mod m, b² mod m, b⁴ mod m, ..., b²ᵏ⁻¹ mod m，并将aⱼ=1的项b²ʲ相乘。
- 时间复杂度: O((log m)² log n)位操作

## 素数理论

### 素数与合数
- **素数**: 大于1且只能被1和自身整除的正整数
- **合数**: 大于1且不是素数的正整数

### 算术基本定理
每个大于1的整数可以唯一地写成一个素数或两个或多个素数的乘积，其中素因子按非递减顺序书写。

### 素数判定方法
1. **方法1**: 测试每个数x<n是否能整除n
2. **方法2**: 测试每个素数x<n是否能整除n  
3. **方法3**: 测试每个素数x≤√n是否能整除n（最优方法）

### 素数无穷性
有无穷多个素数（欧几里得证明）

## 最大公约数与最小公倍数

### 最大公约数 (GCD)
设a和b是整数，不全为0。最大的整数d使得d|a且d|b称为a和b的最大公约数，记作gcd(a,b)。

### 最小公倍数 (LCM)
设a和b是整数。a和b的最小公倍数是能被a和b整除的最小正整数，记作lcm(a,b)。

### 因式分解方法
设a = p₁^a₁ p₂^a₂ ··· pₙ^aₙ 且 b = p₁^b₁ p₂^b₂ ··· pₙ^bₙ
- gcd(a,b) = p₁^min(a₁,b₁) p₂^min(a₂,b₂) ··· pₙ^min(aₙ,bₙ)
- lcm(a,b) = p₁^max(a₁,b₁) p₂^max(a₂,b₂) ··· pₙ^max(aₙ,bₙ)

## 欧几里得算法

### 算法原理
对于两个整数287和91：
1. 287 = 91·3 + 14
2. 91 = 14·6 + 7  
3. 14 = 7·2 + 0
gcd(287,91) = gcd(91,14) = gcd(14,7) = 7

### 正确性证明
**引理**: 设a=bq+r，其中a,b,q和r是整数。那么gcd(a,b)=gcd(b,r)。

**证明**:
- 假设d|a且d|b，则d也整除a-bq=r
- 假设d|b且d|r，则d也整除bq+r=a
- 因此gcd(a,b)=gcd(b,r)

### 时间复杂度
找到gcd(a,b)所需的除法次数是O(log b)，其中a≥b。

## 贝祖定理与扩展欧几里得算法

### 贝祖定理
如果a和b是正整数，那么存在整数s和t使得gcd(a,b)=sa+tb。

### 扩展欧几里得算法
用于找到贝祖恒等式的算法。

**示例**: 将1表示为503和286的线性组合。
503 = 1·286 + 217  
286 = 1·217 + 69  
217 = 3·69 + 10  
69 = 6·10 + 9  
10 = 1·9 + 1  
1 = 10 - 1·9 = 7·10 - 1·69 = 7·217 - 22·69 = 29·217 - 22·286 = 29·503 - 51·286

### 贝祖定理推论
1. 如果gcd(a,b)=1且a|bc，那么a|c
2. 如果p是素数且p|a₁a₂···aₙ，那么p|aᵢ对于某个i

## 同余理论

### 同余的定义
设m是正整数，a,b,c是整数。如果ac≡bc (mod m)且gcd(c,m)=1，那么a≡b (mod m)。

### 模逆
整数ā使得āa≡1 (mod m)被称为a模m的逆。求解线性同余的一种方法使用逆ā（如果存在）。

### 线性同余
形式为ax≡b (mod m)的同余，其中m是正整数，a和b是整数，x是变量。

### 中国剩余定理
解决同余方程组的经典方法，源自中国古代数学问题。

## 特殊素数

### 梅森素数
形式为2ᵖ-1的素数，其中p是素数。
- 2²-1=3, 2³-1=7, 2⁵-1=37, 2⁷-1=127是梅森素数
- 2¹¹-1=2047=23·89不是梅森素数
- 已知的最大素数是梅森素数

## 数论猜想

### 哥德巴赫猜想
每个大于2的偶数n是两个素数的和。

### 孪生素数猜想
有无穷多个孪生素数。
