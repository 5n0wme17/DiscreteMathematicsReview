# 集合论相关内容

## 概述
集合论是数学的基础分支，研究集合及其性质、运算和关系。集合论为现代数学提供了统一的语言和基础。

## 集合的基本概念

### 集合的定义
- **集合**: 具有某种特定性质的事物的总体
- **元素**: 集合中的个体对象
- **表示方法**: 列举法、描述法、文氏图

### 特殊集合
- **空集 (∅)**: 不包含任何元素的集合
- **全集 (U)**: 在特定上下文中所有可能元素的集合
- **有限集**: 元素个数有限的集合
- **无限集**: 元素个数无限的集合

## 集合关系

### 子集与真子集
- **子集**: 如果集合A的所有元素都是集合B的元素，则A是B的子集，记作A⊆B
- **真子集**: 如果A⊆B且A≠B，则A是B的真子集，记作A⊂B
- **性质**: 任何集合都是其自身的子集：S⊆S

### 集合相等
两个集合相等当且仅当每个都是另一个的子集：∀x(x∈A ↔ x∈B)

## 集合的基数

### 基数的定义
- 如果集合S中恰好有n个不同的元素，其中n是非负整数，则称S是有限集
- n是S的基数，记作|S|
- 空集的基数为0：|∅| = 0

## 幂集

### 幂集的定义
- 给定集合S，S的幂集是S的所有子集的集合，记作P(S)
- 如果S是基数为n的集合，则|P(S)| = 2ⁿ

## 元组与笛卡尔积

### 有序n元组
- 有序n元组(a₁,a₂,...,aₙ)是有序集合
- 示例: 二维平面中点的xy坐标

### 笛卡尔积
- A和B的笛卡尔积记作A×B，是所有有序对(a,b)的集合，其中a∈A且b∈B
- A×B = {(a,b) | a∈A ∧ b∈B}
- 性质: A×B ≠ B×A, |A×B| = |A| × |B|

### 多个集合的笛卡尔积
- A₁×A₂×...×Aₙ = {(a₁,a₂,...,aₙ) | aᵢ∈Aᵢ，i=1,2,...,n}
- 笛卡尔积的子集称为从集合A到集合B的关系

## 集合运算

### 基本运算
1. **并集**: A∪B = {x | x∈A ∨ x∈B}
2. **交集**: A∩B = {x | x∈A ∧ x∈B}
3. **补集**: Ā = {x∈U | x∉A}
4. **差集**: A-B = {x | x∈A ∧ x∉B} = A∩Ā

### 不相交集合
- 如果两个集合A和B的交集为空(A∩B=∅)，则称它们为不相交集合
- 不相交集合的并集基数：|A∪B| = |A| + |B|

### 容斥原理
- 两个集合：|E∪F| = |E| + |F| - |E∩F|
- 三个集合：|E∪F∪G| = |E| + |F| + |G| - |E∩F| - |E∩G| - |F∩G| + |E∩F∩G|
- 一般形式：|∪ᵢ₌₁ⁿ Eᵢ| = ∑ₖ₌₁ⁿ (-1)ᵏ⁺¹ ∑₁≤i₁<i₂<···<iₖ≤n |Eᵢ₁∩Eᵢ₂∩···∩Eᵢₖ|

## 集合恒等式

### 基本恒等式
- **恒等律**: A∪∅ = A, A∩U = A
- **支配律**: A∪U = U, A∩∅ = ∅
- **幂等律**: A∪A = A, A∩A = A
- **补集律**: Ā̄ = A

### 运算性质
- **交换律**: A∪B = B∪A, A∩B = B∩A
- **结合律**: A∪(B∪C) = (A∪B)∪C, A∩(B∩C) = (A∩B)∩C
- **分配律**: A∪(B∩C) = (A∪B)∩(A∪C), A∩(B∪C) = (A∩B)∪(A∩C)
- **德摩根定律**: A∩B = Ā∪B̄, A∪B = Ā∩B̄
- **吸收律**: A∪(A∩B) = A, A∩(A∪B) = A
- **补集律**: A∪Ā = U, A∩Ā = ∅

## 广义并集和交集

### 定义
- 集合集合的并集：⋃ᵢ₌₁ⁿ Aᵢ = A₁∪A₂∪...∪Aₙ
- 集合集合的交集：⋂ᵢ₌₁ⁿ Aᵢ = A₁∩A₂∩...∩Aₙ

## 集合的计算机表示

### 位串表示法
- 为全集中的每个元素分配位串中的一个位
- 如果元素在集合中则将位设置为1，否则为0
- 运算对应：
  - 并集: A∨B
  - 交集: A∧B  
  - 补集: Ā

## 函数基础

### 函数的定义
- 从A到B的函数f: A→B，是将B的恰好一个元素分配给A的每个元素的赋值
- 如果b是函数f分配给A的元素a的B的唯一元素，记作f(a)=b
