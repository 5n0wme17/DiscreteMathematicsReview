## 证明定理的方法

证明定理的基本方法：
- 直接证明 - 通过证明如果 p 为真则 q 成立来证明 p→q
- 反证法 - 证明逆否命题 ¬q→¬p
- 矛盾证明 - 证明 (p∧¬q) 与假设矛盾
- 分情况证明 - 对所有可能情况给出证明
- 等价证明 - p↔q 被替换为 (p→q)∧(q→p)

## 大O表示法

设 f 和 g 是从整数集或实数集到实数集的函数。我们说 f(n) = O(g(n))（读作：f(n) 是 g(n) 的大O），如果存在正常数 C 和 k，使得当 n > k 时，|f(n)| ≤ C|g(n)|。

## 数学归纳法

典型的数学归纳法证明，证明对于所有整数 n ≥ b，陈述 P(n) 为真，包括三个步骤：

1. 我们证明 P(b) 为真。 - 基础步骤

2. 然后，对于所有 n > b，我们证明要么 (*) P(n-1) → P(n) 要么 (**) P(b)∧P(b+1)∧···∧P(n-1) → P(n)

我们需要做出 P(n-1) 或 P(b)∧P(b+1)∧···∧P(n-1) 的归纳假设。然后我们使用 (*) 或 (**) 推导出 P(n)。

3. 我们根据数学归纳法原理得出结论，对于所有 n ≥ b，P(n) 为真。
